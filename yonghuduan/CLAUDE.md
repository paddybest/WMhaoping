## 微信小程序开发注意事项

### WXML 限制
- **问题**：WXML 不支持直接调用数组方法如 `indexOf()`、`includes()` 等
- **症状**：条件判断始终返回 `false`，样式类无法正确添加
- **解决方案**：使用 WXS 模块
  1. 创建 `index.wxs` 文件提供 `contains()` 函数
  2. 在 WXML 中引入：`<wxs src="index.wxs" module="tools" />`
  3. 使用：`{{tools.contains(selectedTags, item) ? 'selected' : ''}}`

### 标签选择功能修复经验
- **问题**：标签点击后没有紫色选中效果
- **根本原因**：WXML 中 `indexOf()` 方法不被支持
- **解决步骤**：
  1. 创建 WXS 模块处理数组包含检查
  2. 修复 class 绑定语法
  3. 确保选中状态样式正确应用

### CSS 样式建议
- 使用 `!important` 确保样式优先级
- 简化选择器，避免过度复杂
- 移除不必要的伪元素（::before、::after）
