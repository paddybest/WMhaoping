## 📝 项目简介

好评宝用户端小程序是基于微信小程序开发框架构建的消费者应用，为用户提供产品浏览、扫码抽奖、评价管理等功能。小程序通过API与后端服务交互，实现了多租户架构，每个用户只能访问指定商家的数据。

### 核心功能

- 🏪 **商家选择** - 优雅的商家选择界面
- 📦 **产品浏览** - 产品列表、详情、图片查看
- 🏆 **奖品查看** - 奖品展示、抽奖说明
- 🎟 **抽奖功能** - 扫码抽奖、核销
- ⭐ **评价系统** - 商品评价、星级评分
- 📊 **扫码统计** - 扫码记录、访问统计
- 🎞 **客服接入** - 在线客服一键接入
