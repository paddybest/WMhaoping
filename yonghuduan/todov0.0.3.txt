抽奖界面技术需求文档
一、项目背景
当前AI评价助手小程序的抽奖功能在结果页(result)中以简单弹窗形式展示奖励码，缺乏互动性和视觉吸引力。为提升用户体验和参与感，需要设计一个独立的抽奖页面，通过动画效果和增强的交互流程来展示奖励。

二、设计目标
创建沉浸式抽奖体验 - 独立页面展示抽奖流程
增强用户互动 - 通过动画和音效提升参与感
清晰的用户引导 - 明确的奖励领取流程
无缝集成 - 与现有奖励码系统无缝对接
三、技术方案概述
3.1 核心实现思路
当前流程: result页 → 弹窗显示奖励码 → 用户复制
新流程: result页 → 抽奖页 → 转盘动画 → 结果展示 → 用户操作

5.3 抽奖流程代码示例

// 开始抽奖
async onSpinButtonClick() {
  if (this.data.isSpinning) return

  this.setData({ isSpinning: true, lotteryStatus: 'spinning' })

  // 计算旋转角度(必定停在奖励码位置)
  const baseRotation = 1800  // 5圈
  const targetAngle = 180    // 奖励码位置
  const randomOffset = Math.floor(Math.random() * 20) - 10
  const rotation = baseRotation + targetAngle + randomOffset

  this.setData({ currentRotation: rotation })

  // 等待动画完成
  await new Promise(resolve => setTimeout(resolve, 3500))

  // 展示结果
  this.showLotteryResult()
}

// 展示结果
showLotteryResult() {
  this.setData({
    isSpinning: false,
    lotteryStatus: 'result',
    showConfetti: true
  })

  // 震动反馈
  wx.vibrateShort({ type: 'heavy' })

  // 3秒后清除彩带
  setTimeout(() => {
    this.setData({ showConfetti: false })
  }, 3000)
}

// 复制奖励码
onCopyCode() {
  wx.setClipboardData({
    data: this.data.rewardCode,
    success: () => {
      wx.showToast({ title: '奖励码已复制', icon: 'success' })
      this.setData({ lotteryStatus: 'claimed' })
    }
  })
}

async getRewardCode() {
  try {
    wx.showLoading({ title: '生成奖励中...' })

    const app = getApp()
    const res = await wx.cloud.callFunction({
      name: 'createLotteryCode',
      data: {
        openid: app.globalData.openid || wx.getStorageSync('openid')
      }
    })

    wx.hideLoading()

    if (res.result.success) {
      const rewardCode = res.result.data.code

      // 修改:跳转到抽奖页面
      wx.navigateTo({
        url: `/pages/lottery/index?code=${rewardCode}&from=result`,
        fail: (err) => {
          // 降级方案:跳转失败显示弹窗
          console.error('跳转失败:', err)
          this.setData({ rewardCode, showModal: true })
        }
      })
    }
  } catch (error) {
    wx.hideLoading()
    wx.showToast({ title: '获取奖励失败', icon: 'none' })
  }
}
