# AI 评价助手 - 数据库与云存储配置指南

## 概述
本文档指导您完成 AI 评价助手小程序的数据库和云存储配置，包括创建集合、索引配置、权限设置以及示例数据导入。

## 前提条件
- 已安装微信开发者工具
- 项目已配置云开发环境 (env: cloud1-9gbrkqwy4f67587b)
- 项目已上传云开发

## 第一阶段：创建数据库集合

### 1.1 打开云开发控制台
1. 在微信开发者工具中，点击右上角工具栏的【云开发】按钮
2. 选择环境：`cloud1-9gbrkqwy4f67587b`
3. 进入【数据库】管理

### 1.2 创建 products 集合（类目配置表）
1. 点击【+】号创建集合

### 1.3 创建 images 集合（素材库）
1. 再次点击【+】号创建集合
2. 集合名称：`images`
3. 权限设置：自定义（稍后统一配置）

### 1.4 创建 lottery_codes 集合（奖励记录）
1. 再次点击【+】号创建集合
2. 集合名称：`lottery_codes`
3. 权限设置：自定义（稍后统一配置）

## 第二阶段：配置索引

### 2.1 为 lottery_codes 集合创建唯一索引
1. 进入 `lottery_codes` 集合
2. 点击【索引】标签
3. 点击【+】创建索引
4. 字段名：`code`
5. 索引类型：唯一索引
6. 索引名称：`unique_code`

### 2.2 为 images 集合创建索引
1. 进入 `images` 集合
2. 点击【索引】标签
3. 点击【+】创建索引
4. 字段名：`categoryId`
5. 索引类型：普通索引
6. 索引名称：`category_index`

## 第三阶段：云存储配置

### 3.1 创建云存储目录
1. 在云开发控制台中，点击【存储】管理
2. 确保当前环境是 `cloud1-9gbrkqwy4f67587b`
3. 如果没有根目录，请创建一个基础目录结构
4. 创建 `review-images` 目录：
   - 点击【+】新建目录
   - 目录名称：`review-images`

### 3.2 上传图片素材
根据类目上传对应的商品图片素材：

#### 女装目录
- 创建子目录：`review-images/female-clothes`
- 上传女装相关的商品图片（至少3张）
- 记录每张图片的 fileID

#### 男装目录
- 创建子目录：`review-images/male-clothes`
- 上传男装相关的商品图片（至少3张）
- 记录每张图片的 fileID

#### 数码产品目录
- 创建子目录：`review-images/digital`
- 上传数码产品相关的图片（至少3张）
- 记录每张图片的 fileID

#### 美妆护肤目录
- 创建子目录：`review-images/beauty`
- 上传美妆护肤相关的图片（至少3张）
- 记录每张图片的 fileID

### 3.3 配置存储权限
1. 在存储管理页面，点击【权限设置】
2. 为 `review-images` 目录设置权限：
   - 所有用户可读
   - 仅管理员可写

## 第四阶段：数据库权限配置

### 4.1 全局权限设置
1. 回到数据库首页，点击【权限设置】
2. 设置全局默认权限：
   - 自定义权限
   - 禁止所有用户读写
   - 仅允许通过云函数访问

### 4.2 各集合权限配置

#### products 集合权限
- 读权限：所有人可读
- 写权限：仅管理员可写

#### images 集合权限
- 读权限：所有人可读
- 写权限：仅管理员可写

#### lottery_codes 集合权限
- 读权限：仅创建者可读
- 写权限：仅管理员可写

## 第六阶段：验证配置

### 6.1 数据库验证
1. 进入每个集合，检查数据是否正确
2. 测试索引是否生效（可以在云函数中测试查询）
3. 验证权限设置是否符合要求

### 6.2 云存储验证
1. 检查 review-images 目录是否创建成功
2. 验证图片是否能正常访问
3. 确认权限设置是否生效

### 6.3 测试连接
使用以下云函数测试数据库连接：
```javascript
// 测试数据库连接
const testDB = async () => {
  const res = await wx.cloud.database().collection('products').get()
  console.log('数据库连接成功:', res.data)
}
```

## 注意事项

1. **安全第一**：确保数据库权限设置正确，避免数据泄露
2. **备份重要**：定期备份数据库数据
3. **性能优化**：合理的索引设置能提升查询性能
4. **文件管理**：定期清理不需要的图片文件
5. **监控使用**：监控云存储使用量，避免超出配额

## 常见问题

### Q: 如何获取图片的 fileID？
A: 图片上传后，在云存储管理页面点击图片，可以在详情页看到 fileID

### Q: 如何修改权限？
A: 在每个集合或存储目录的权限设置页面可以修改权限

### Q: 如何备份数据？
A: 在云开发控制台的数据库管理页面，可以使用导出功能备份数据
