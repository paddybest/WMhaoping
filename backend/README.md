## 📝 项目简介

好评宝后端服务是一个完整的商家管理系统后端，为商家端和用户端小程序提供API接口。系统支持多租户架构，每个商家拥有独立的数据隔离，确保数据安全和隐私。

### 核心功能

- 🏪 **商家管理** - 注册、登录、信息管理
- 📦 **产品管理** - 商品、分类、图片、标签管理
- 🏆 **奖品管理** - 奖品创建、库存管理、概率设置
- 🎟 **抽奖系统** - 抽奖码生成、核销、统计分析
- 📊 **数据分析** - 用户行为分析、扫码统计、评价统计
- 🛡️ **安全防护** - 多租户隔离、跨商家访问防护、二维码防伪

## 🚀 快速开始

### 环境要求

\`\`\`bash
# 1. 安装Node.js 18+
node --version  # >= 18.0.0

# 2. 安装MySQL 8.0+
# 或使用Docker
docker --version  # >= 20.10.0

# 3. 安装Redis 7+
redis-cli --version  # 或使用Docker

# 4. 克隆项目
git clone <repository-url>
cd haopingbao/backend
