## 📝 项目简介

好评宝商家端应用是基于React的商家管理系统，为商家提供完整的产品管理、分类管理、奖品管理、抽奖营销等功能。应用通过API与后端服务交互，实现了多租户架构支持。

### 核心功能

- 📦 **产品管理** - 商品、分类、图片、标签
- 🏆 **奖品管理** - 奖品创建、库存管理、抽奖设置
- 🎟 **抽奖营销** - 抽奖码生成、核销、使用统计
- 📊 **数据分析** - 销售统计、用户评价、扫码分析
- 🏪 **商家管理** - 店铺信息、二维码、客服设置
- 📞 **客服系统** - 在线客服接入
- 🛡️ **安全** - 商家认证、权限控制
